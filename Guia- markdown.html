Guia de Markdown

O que é Markdown?

Markdown é uma linguagem de marcação leve para deixar seu texto bonito e organizado sem esforço.

1. Cabeçalhos

Como escrever:

# Cabeçalho 1
## Cabeçalho 2
### Cabeçalho 3

Como aparece:

Cabeçalho 1

Cabeçalho 2

Cabeçalho 3

2. Negrito e Itálico

Sintaxe:

*Itálico*
**Negrito**
***Negrito e Itálico***

Visualização:

Itálico

Negrito

Negrito e Itálico

3. Listas

Não ordenadas:

- Item 1
- Item 2
  - Subitem 2.1

Ordenadas:

1. Primeiro
2. Segundo
3. Terceiro

Visualização:

  • Item 1
  • Item 2
    • Subitem 2.1
  1. Primeiro
  2. Segundo
  3. Terceiro

4. Links e Imagens

Link:

[Google](https://www.google.com)

Imagem:

![Exemplo](https://via.placeholder.com/150)

Visualização:

Google

5. Código

Sintaxe:

`Código em linha`
```
Bloco de código
Linha 2
```

Visualização:

Código em linha

Bloco de código
Linha 2

6. Citação

Sintaxe:

> Esta é uma citação

Visualização:

Esta é uma citação
